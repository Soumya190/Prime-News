# Prime-News
A responsive news platform with category-based filtering and keyword search, built using Next and SASS.

This News Portal Website is built to deliver real-time news updates across multiple categories. It allows users to browse news by topic and search for articles.
The site is fully responsive, ensuring a seamless experience across desktop, tablet, and mobile devices.

## Features
Category-Based Filtering – Quickly find news in specific topics.
Keyword Search – Search articles by keywords.
Responsive Design – Optimized for all screen sizes.

## Tech Stack
Nextjs, CSS, SASS
API: NewsAPI.org for fetching latest news
Hosting: netlify
